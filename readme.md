# ✨ Lumina

O **Lumina** é uma aplicação web desenvolvida para auxiliar estudantes na preparação para vestibulares e ENEM, oferecendo uma interface moderna e intuitiva para apoiar o aprendizado.

---

## 📚 Sobre o Projeto

O Lumina tem como objetivo ajudar estudantes a organizarem seus estudos, revisarem conteúdos e tirarem dúvidas de forma rápida e acessível.

A aplicação utiliza uma interface baseada em chat, proporcionando uma experiência simples, familiar e eficiente para o usuário.

---

## 🎯 Problema

Estudantes frequentemente enfrentam dificuldades como:

* Falta de organização nos estudos
* Dificuldade em revisar conteúdos
* Falta de acesso rápido a explicações

---

## 💡 Solução

O Lumina propõe uma plataforma onde o usuário pode interagir com um assistente educacional através de uma interface de chat, facilitando o processo de aprendizado.

---

## 👥 Público-alvo

* Estudantes do ensino médio
* Alunos de cursos técnicos
* Pessoas que estão se preparando para vestibulares e ENEM

---
